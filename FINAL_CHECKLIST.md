## 🐛 常见问题快速修复

### 问题1: Cannot find module
```bash
rm -rf node_modules pnpm-lock.yaml
pnpm install
```

### 问题2: 端口被占用
修改 `vite.config.js`:
```javascript
server: {
  port: 3003, // 改为其他端口
}
```

### 问题3: 样式不显示
检查 CSS 导入路径：
```javascript
import '@ldesign/progress-core/src/styles/index.css';
```

### 问题4: 热更新不工作
```bash
# 重启服务
Ctrl+C
pnpm dev:xxx
```
